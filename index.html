The Grid

A single page application built using React.js. This application is designed for video game players to have a way of organizing social interactions and connecting with one another either online or in person. Users can add friends, create posts, comment on other posts, join clans and more! Mongo DB is the database that is used to store all the profile and user information while Heroku is the platform that hosts the application. User credential validation is important for use of the site, because the various different API endpoints are restricted to requests made that contain a valid JSON web token.

Next Game

A front end application that utilizes Firebase as the means for user authentication. This project is a means for clients to be reccomended video games based on their preferences. It uses Rawg.io as well as IGDB's API. There was no back end code written for this project. User's search history and wishlist is stored within their local storage.

Crop Circle

CropCircle is a fully functioning message board. The front end is rendered using Express-Handlebars.js as the template engine and Sequel as the database. CropCircle adheres to the REST constraints in order to make it easily scalable. Clients can create accounts and subsequently create posts and comments on existing posts within different message boards on the site. If a board does not exist, then the user can simply create it by going to the URL endpoint of their choosing. Similar to how Reddit works.

BAMazon

A command line interface that simulates a retail store. BAMazon consits of three different modules and each have different functions. Users can simulate a customer, manager or supervisor. The modules control buying products, ordering products and creating new inventory and viewing the sales and overhead of the different departments.

Friend Finder

A site that mimics a dating site or a site meant to meet new friends. Users answer questions and the website matches them with the friend that is the most compatible match. Friend Finder does not use a database; however, it does have memory by being a server hosted on Heroku and storing user responses and data in an array for reference.

Liri Node App

Liri is a command line interface that uses three seperate APIs in order to do different functions based on the user's whims. Liri is able to look up movie information using IMDB's API, concert information using Bands in Town's API and song information using Spotify's API. Liri can take in node arguments from the command line or if no arguments are input by the user there is a prompt that will appear to help the user use Liri.
